---

## 📖 À propos

Skaymog est une plateforme communautaire dédiée aux passionnés d'animés. Elle permet de :

- 📚 **Consulter une bibliothèque de 579 animés** (Shonen, Seinen, Shojo, Isekai, Romance, etc.)
- ❤️ **Liker et recommander** vos animés préférés
- 👀 **Marquer les animés comme "vus"** et gérer vos listes personnelles
- 💬 **Commenter et échanger** des avis sur chaque animé
- 👥 **Suivre d'autres utilisateurs** et découvrir leurs recommandations
- 💬 **Messagerie en temps réel** (texte + audio) entre amis
- 🔔 **Notifications** pour les commentaires, messages et propositions validées
- 🛠️ **Panel administrateur** pour gérer la bibliothèque (ajout, modification, suppression)
- 🌗 **Thème clair/sombre** et **multi-langues** (FR, EN, ES, JA)

---

## 🚀 Fonctionnalités principales

| Fonctionnalité | Description |
|---------------|-------------|
| **Bibliothèque** | 579 animés classés par catégories avec recherche et filtres |
| **Fiches détaillées** | Résumé, épisodes, studio, saison, likes, commentaires |
| **Listes personnalisées** | "Vus", "À voir", "Aimés" + listes sur mesure |
| **Recommandations** | Recommandez un animé et voyez ceux des autres |
| **Système d'amis** | Abonnez-vous et devenez amis en retour (suivi mutuel) |
| **Messagerie** | Chat en direct avec messages texte et audio |
| **Notifications** | Réactions aux commentaires, messages, propositions acceptées |
| **Admin** | Gestion complète des animés (CRUD) + code admin dynamique |
| **Thème** | Clair / Sombre avec sauvegarde automatique |
| **Langues** | Français, Anglais, Espagnol, Japonais |

---

## 🛠️ Stack technique

- **Frontend** : HTML5, CSS3, JavaScript (Vanilla)
- **Framework** : Vue.js 3 (CDN)
- **Stockage** : LocalStorage (persistance des données)
- **Hébergement** : GitHub Pages
